## 6. –ö–æ–Ω—Ç—Ä–æ–ª—å–Ω—ã–µ –≤–æ–ø—Ä–æ—Å—ã

### 6.1. –°—Ñ–æ—Ä–º—É–ª–∏—Ä—É–π—Ç–µ –æ—Å–Ω–æ–≤–Ω–æ–µ —Å–≤–æ–π—Å—Ç–≤–æ –±–∏–Ω–∞—Ä–Ω–æ–≥–æ –¥–µ—Ä–µ–≤–∞ –ø–æ–∏—Å–∫–∞ (BST).
–í BST –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —É–∑–ª–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ø—Ä–∞–≤–∏–ª–æ:

–≠—Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –ª–æ–≥–∞—Ä–∏—Ñ–º–∏—á–Ω–æ–≥–æ –ø–æ–∏—Å–∫–∞.

### 6.2. –û–ø–∏—à–∏—Ç–µ –∞–ª–≥–æ—Ä–∏—Ç–º –≤—Å—Ç–∞–≤–∫–∏ –Ω–æ–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ BST. –ö–∞–∫–æ–≤–∞ —Å–ª–æ–∂–Ω–æ—Å—Ç—å —ç—Ç–æ–π –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤ —Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –∏ –≤—ã—Ä–æ–∂–¥–µ–Ω–Ω–æ–º –¥–µ—Ä–µ–≤–µ?
**–ê–ª–≥–æ—Ä–∏—Ç–º –≤—Å—Ç–∞–≤–∫–∏:**

**–°–ª–æ–∂–Ω–æ—Å—Ç—å:**

### 6.3. –ß–µ–º –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –æ–±—Ö–æ–¥ –¥–µ—Ä–µ–≤–∞ –≤ –≥–ª—É–±–∏–Ω—É (DFS) –æ—Ç –æ–±—Ö–æ–¥–∞ –≤ —à–∏—Ä–∏–Ω—É (BFS)? –ù–∞–∑–æ–≤–∏—Ç–µ –≤–∏–¥—ã DFS-–æ–±—Ö–æ–¥–æ–≤.

* **BFS (Breadth-First Search)** ‚Äî –æ–±—Ö–æ–¥–∏—Ç –¥–µ—Ä–µ–≤–æ –ø–æ —É—Ä–æ–≤–Ω—è–º, —Å–ª–æ–π –∑–∞ —Å–ª–æ–µ–º.
